# RODB - Ragnarok Online Database

RODB é um projeto de estudo desenvolvido em PHP para criar uma base de dados de itens e cartas do Ragnarok Online.

O projeto utiliza os arquivos YAML do rAthena como fonte de dados e os transforma em uma interface web para consulta e visualização das cartas.

## Tecnologias

- PHP
- Composer
- Symfony YAML
- HTML
- CSS

## Funcionalidades atuais

- Leitura de arquivos YAML do rAthena
- Filtragem de itens do tipo `Card`
- Exibição dinâmica das cartas em HTML
- Exibição de informações como:
  - Nome
  - ID
  - AegisName
  - Script/bônus
